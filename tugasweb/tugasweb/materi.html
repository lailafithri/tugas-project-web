MATERI 1


Definisi Internet

Internet (kependekan dari interconnection-networking) adalah seluruh jaringan komunikasi yang menggunakan media elektronik, yang saling terhubung menggunakan standar sistem global Transmission Control Protocol/Internet Protocol Suite (TCP/IP) sebagai protokol pertukaran paket (packet switching communication protocol) untuk melayani miliaran pengguna di seluruh dunia.Rangkaian jaringan yang terbesar dinamakan Internet. Cara menghubungkan rangkaian dengan kaidah ini dinamakan internetworking ("antarjaringan").

Apa itu Web? Layanan internet yang berjalan pada protokol aplikasi HTTP

Perbedaan Aplikasi berbasis Web dengan Aplikasi berbasis Internet

Aplikasi Berbasis Web

  • Menggunakana protokol HTTP
  • Aplikasi di sisi server berkomunikasi dengan client melalui web server
  • Aplikasi di sisi client umumnya berupa web browser

Aplikasi Berbasis Internet

  • Menggunakan protokol aplikasi yang sudah ada atau mendefinisikan protokol sendiri
  • Aplikasi di sisi server berkomunikasi langsung dengan client
  • Aplikasi di sisi client dapat berupa aplikasi yang berdiri sendiri atau ditempelkan dalam aplikasi lain

MATERI 2 CLIENT SIDE PROGRAMMING


Client side programming biasanya lebih banyak dikerjakan olehfront-end developer. Adapun komponennya : HTML,CSS, dan Javascript.

HTML (HyperText Markup Language) adalah standard bahasa yang digunakan untuk menampilkan dokumen web. Fungsinya Mengontrol tampilan konten, Memuat form untuk transaksi data, Memuat obyek lain seperti image, audio, video,dll.

HTML TOOLS : 1.Browser (Software di client untuk menerjemahkan tag HTML) contohnya : IE, Chrome, Firefox, Opera, dll. 2.Editor (Program untuk menulis tag HTML) contohnya : Notepad, Notepad++, Sublime, Dreamweaver, dll.

Di bahasa pemrograman pada umumnya biasanya kita mengenal code, pada HTML kita mengenal tag. Contoh tag: body,head,div,img.

Struktur document HTML bisa di bagi mejadi tiga bagian utama:html, head, dan body.Di dalam fungsi head bagian ini biasanya dimuat tag title yang menampilkan judul dari halaman web.Tag body di gunakan untuk menampilkan text, image link dan semua yang akan di tampilkan pada web page.

Block level element: terdapat 6 tingkatan, yaitu H1 sampai H6. Ada juga paragraf untuk Menampilkan teks dalam bentuk paragraf diartikan dalam huruf p. kemudian br untuk memulau baris baru, Font untuk menentukan format tampilan font, dan Color sebagai attribute dengan 3 kategori warna (RGB).

MATERI 4 CSS


CSS (Cascading Style Sheet) adalah salah satu bahasa desain web (style sheet language) yang mengontrol format tampilan sebuah halaman web yang ditulis dengan menggunakan penanda(markup laguage. Biasanya CSS digunakan untuk mendesain sebuah halaman HTML dan XHTML, tetapi sekarang CSS bisa diaplikasikan untuk segala dokumenXML, termasuk SVG dan XUL bahkan ANDROID.

CSS berfungsi mendefenisikan style untuk suatu teks dengan jenis huruf,ukuran,warna tertentu.

Adapun metode pembuatan CSS : 1. Style Sheet Inline (mengetikkan langsung dalam tag html sebagai atribut), 2. Style Sheet Internal (Menggunakan tag style di dalam tag head), 3. Style Sheet External (Menyimpan informasi style ke dalam sebuah file dengan ekstensi/type file .css).
